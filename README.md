# E-Commerce Data Analysis

This project analyzes E-Commerce product sales data using Python, Pandas, NumPy, and Matplotlib.

## Features

- Data cleaning using Pandas
- Missing value handling
- NumPy calculations
- Sales data analysis
- Data visualization using Matplotlib
- Colorful bar chart visualization

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Excel Dataset

## Files Included

- ecommerce_dataset.xlsx
- ecommerce_data_cleaning.py
- cleaned_ecommerce_dataset.xlsx
- ecommerce_visualization.py
- ecommerce_bar_chart.png

## Output

The project generates:
- Cleaned dataset
- Sales analysis
- Product sales visualization graph
